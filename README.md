هذا هو حلي لتحدي بناء صفحة شخصية من موقع **Frontend Mentor**. يركز المشروع على تقديم سيرة ذاتية تقنية بشكل عصري ومتجاوب مع جميع الشاشات.

- **مستودع الكود (Repository):** [عرض الكود على GitHub](https://github.com/GamePatchHub88/Personal-Website)

## 🛠️ التقنيات المستخدمة (Tech Stack)

لقد اعتمدت في هذا المشروع على أحدث تقنيات الـ Front-End لضمان كود نظيف وقابل للتطوير:

* **HTML5:** استخدام العناصر الدلالية (Semantic HTML) لتحسين محركات البحث (SEO).
* **Sass (SCSS):** لتنظيم الأنماط البرمجية باستخدام المتغيرات (Variables) والـ Mixins والـ Nesting.
* **CSS Grid:** لبناء الهيكل الرئيسي للصفحة وتوزيع الأقسام بدقة.
* **CSS Flexbox:** لتنسيق العناصر الداخلية ومحاذاة النصوص والأيقونات.
* **Mobile-First Workflow:** التصميم بدأ من الشاشات الصغيرة وصولاً إلى شاشات الكمبيوتر لضمان أفضل تجربة مستخدم.

## 🚀 المميزات

- **Responsive Design:** الموقع متوافق تماماً مع الجوال، الجهاز اللوحي، والحاسوب.
- **Clean Architecture:** فصل ملفات Sass لتسهيل القراءة والصيانة.
- **Hover Effects:** إضافة تأثيرات تفاعلية ناعمة على الأزرار والروابط.
